# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built for The Odin Project assignment.

## Overview

This project implements the classic Rock Paper Scissors game with a clean user interface, interactive gameplay, and score tracking. The game allows one player to compete against the computer and displays the result after each round.

## Features

- Player vs. computer gameplay
- Random computer choices
- Round results displayed in real time
- Score tracking for player and computer
- Reset game functionality

## Installation

1. Clone this repository:
   ```bash
   git clone <repository-url>
   ```
2. Open `index.html` in your browser.

## Usage

- Click one of the options: Rock, Paper, or Scissors.
- The computer will make a random choice.
- The round winner will be displayed and the scores updated.
- Use the reset button to start a new game.

## Project Goals

- Practice DOM manipulation with JavaScript
- Use event listeners to handle user interactions
- Implement game logic and random choice generation
- Build a responsive and user-friendly UI

## Notes

This project is part of The Odin Project curriculum and is intended as a beginner-friendly JavaScript exercise.
